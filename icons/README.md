# Ícones do App

Esta pasta deve conter os seguintes arquivos de ícone:

- `icon-192.png` - 192x192 pixels
- `icon-512.png` - 512x512 pixels  
- `apple-touch-icon.png` - 180x180 pixels

## Como Criar os Ícones

### Opção 1: Gerador Online (Recomendado)

1. Acesse: https://realfavicongenerator.net/
2. Faça upload de uma imagem (pelo menos 512x512)
3. Configure as opções
4. Baixe os arquivos gerados
5. Coloque nesta pasta

### Opção 2: Criar Manualmente

Use qualquer editor de imagens (Photoshop, GIMP, Canva, etc.):

1. Crie uma imagem quadrada com fundo verde (#22c55e)
2. Adicione um ícone ou texto (ex: "✓" ou "Metas")
3. Exporte nos tamanhos necessários
4. Coloque nesta pasta

### Opção 3: Usar o SVG Base

Há um arquivo `icon-base.svg` nesta pasta que você pode:
1. Abrir em um navegador
2. Fazer screenshot
3. Redimensionar para os tamanhos necessários

## Ícone Temporário

Se você quiser testar o app imediatamente sem criar ícones, pode usar qualquer imagem PNG nos tamanhos corretos. O app funcionará, mas o ícone na tela inicial será genérico.
